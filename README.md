# auv2018

Uses ROS Lunar for Ubuntu 17.04 
and python 2.7.13

## First time setup:

### In terminal:
- git clone Robosub
- cd Robosub/
- python robosub.py

### In the auv program:
- if ROS has not been installed, it will be installed automatically
- After installing restart computer
- After restart cut and paste the Robosub folder into '~/catkin_ws/src/'

### In terminal:
- cd ~/catkin_ws
- catkin_make
